#자바 기반 다중 클라이언트 채팅 및 파일 전송 프로그램
본 프로젝트는 자바 기반 서버-클라이언트 구조를 활용한 다중 사용자 채팅 및 파일 전송 프로그램입니다.


##주요 기능
다중 클라이언트 채팅
GUI를 통한 메시지 입력 및 파일 선택
클라이언트 간 파일 전송(서버를 거쳐 전달)
멀티쓰레드 기반 서버 구현

##프로젝트 구조
ChatClient.java: 클라이언트 코드
ChatServer.java: 서버 코드
README.md: 프로젝트 설명

##실행방법 
1. 서버 실행
ChatServer.java 파일을 먼저 실행
콘솔에 Server started 메시지 확인
2. 클라이언트 실행
각 클라이언트마다 ChatClient.java 실행
파일 안에 있는 SERVER_IP를 서버가 켜진 PC 주소로 바꾼다.
여러 클라이언트를 동시에 실행하면 채팅과 파일 전송을 테스트할 수 있다.
3. 기본 사용법
메시지를 입력하고 전송 버튼 클릭 --> 채팅 가능
ftp 버튼 클릭 후 파일 선택 --> 서버를 통해 다른 클라이언트로 파일 전송 가능(파일목록을 통해 확인 가능)

##참고 사항
서버와 클라이언트는 같은 네트워크 안에서만 연결 가능 
SERVER_IP는 예시로 userIP로 되어있으니, 실행 시 서버 PC IP로 바꿔야 한다.
여러 클라이언트를 동시에 켤 수 있다.
